# Aqua Sports Simulator (Rust Edition)

Java Swingで書かれたレトロなミニゲームを、Rustの軽量2Dゲームエンジン `macroquad` を使用して移植・拡張した作品です。

## 🎮 ゲームモード
1. **水泳（Swimming）**: スペースキーを連打してCPUと競争します。スタミナ管理が鍵！
2. **飛び込み（Diving）**: タイミングよくスペースキーを押して空中で回転を決めます。

## 🛠️ 動作方法
Rustがインストールされた環境で以下を実行してください。
```bash
cargo run
